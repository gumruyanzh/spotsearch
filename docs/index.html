Features

Everything you need to find files fast.

⚡

Instant Search

Blazing-fast file search powered by macOS native mdfind. Results appear as you type.

⌨️

Option + Space

Global hotkey launches SpotSearch from anywhere. No need to switch apps or click around.

💻

Menu Bar App

Lives quietly in your menu bar. Always one click or keystroke away, never in your dock.

🔍

Smart Filters

Filter results by file type: Documents, Images, PDFs, Code, Audio, Video, and more.

📎

Extension Filter

Narrow results to a specific file extension like .swift, .tsx, or .json for precise searches.

👁️

Quick Preview

Press Space to preview any file with macOS Quick Look without opening it.

📋

Clipboard History

Access your recent clipboard entries and paste them with a single click. Never lose a copied snippet.

Keyboard Shortcuts

Designed for keyboard-first workflows.

Shortcut Action
⌥ + Space Open / close SpotSearch
↑ / ↓ Navigate results
⏎ Open selected file
Space Quick preview with Quick Look
⌘ + ⏎ Reveal in Finder
⌘ + C Copy file path to clipboard
Esc Close window
